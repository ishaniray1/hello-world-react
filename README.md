# Hello World · React

A minimal, clean React web app built with [Vite](https://vitejs.dev/).

## Features
- Personalized greeting input
- Interactive counter
- Dark-themed, responsive UI

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack
- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
